Index terms

Index terms should not start with the product name, for example if the product name is oXygen, do not use

oxygen validation

as index term, use simply

validation

without mentioning the product name because that will not be relevant, as the product is already the context for the index term.

Rule
restrictNesting

indexterm
maxNestingLevel
1
message
indexterms
Rule
avoidWordInElement

oXygen
message
Do not use oXygen in index terms!

Titles and short descriptions

Titles should not be very long, we should keep them less than 1 words.

Titles should not be long, keep them less than 10 words.

Short descriptions should be less than 20 words and also less than 140 characters so we can use them easily with twitter.
